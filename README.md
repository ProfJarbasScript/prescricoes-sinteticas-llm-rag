# prescricoes-sinteticas-llm-rag
As prescrições foram geradas sinteticamente com base em modelos hospitalares reais, simulando diferentes cenários clínicos de forma totalmente anônima e sem uso de dados pessoais.
A criação das prescrições ocorreu por meio de um modelo base desenvolvido manualmente, expandido com o auxílio do ChatGPT (GPT-5), que foi utilizado para gerar variações realistas e diversificadas de prescrições médicas.
O objetivo deste conjunto de dados é fornecer uma base controlada e segura para o teste de modelos de linguagem (LLMs) combinados à técnica de Geração Aumentada por Recuperação (RAG), aplicados à detecção de Problemas Relacionados a Medicamentos (PRMs), como dosagens incorretas, interações medicamentosas, vias de administração inadequadas e prescrições incompletas.

PROMPT: Gere uma prescrição médica hospitalar sintética em português, seguindo o exemplo colocado nesta conversa, com informações ficticias, e 3 a 5 medicamentos comuns. Use dosagens realistas e linguagem técnica padronizada. Não inclua dados pessoais reais.
